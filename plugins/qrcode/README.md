# SliDesk Plugin: QRCode

With this plugin you can generate a QR Code from a given url/text

In your slides, add `<div class="sd-qrcode" data-url="URL OR TEXT" style="width: 300px"></div>`.

If you want to add an icon in the center of the qrcode:

`<div class="sd-qrcode" data-url="URL OR TEXT" style="width: 300px" data-icon="/favicon.svg"></div>`.
