# Split

This template follows this structure:

```
## Title of the slide .[#@gouz__split/split]

[[left]]

Left part

[[/left]]


and the rest of then content


[[right]]

Right part

[[/right]]

and it can continue here
```

And will render the slide like this:

```
        Title of the slide

Left part         |  Right part
                  |  and rest of content
                  |  and it can continue here
```
