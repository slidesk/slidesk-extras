# Template: MidCoverRight

This template follows this structure:

```
## Title of the slide .[#mid-cover-right]

[[img]]

!image(assets/...)

[[/img]]

and the rest of then content
```

And will render the slide like this:

```
                              |
       Title of the slide     | IMG (covered)
 and the rest of then content |
                              |
```
